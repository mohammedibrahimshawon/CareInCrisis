Important FAQ

  1. What is the Document Object Model?
    The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.
  2. How do you select an element from the DOM?
    You can select elements using methods such as `getElementById()`, `getElementsByClassName()`, `getElementsByTagName()`, `querySelector()`, and `querySelectorAll()`.
  3. What is event delegation in the context of the DOM, and why is it useful?
    Event delegation is a technique where you add an event listener to a parent element instead of multiple child elements. This is useful because it improves performance and ensures that dynamically added elements also trigger the event.
